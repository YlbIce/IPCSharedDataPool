## 一、项目概述

### 1.1 背景

当前 `IEC61850Mapper` 模块实现了基础的 IEC61850 数据模型映射功能，主要依赖共享内存进行数据交换。通过引入 `libiec61850` 库，可以实现完整的 IEC61850 协议栈功能，包括 MMS 通信、GOOSE/SV 实时通信、报告服务等。

### 1.2 目标

1. **完整的 IEC61850 服务器能力**：支持标准 MMS 协议栈
2. **实时通信支持**：GOOSE 发布/订阅、SV 采样值
3. **SCL 配置文件支持**：标准 IED 配置文件解析
4. **报告服务**：缓冲/非缓冲报告控制块
5. **日志与 SOE**：符合 IEC61850 标准的事件记录
6. **高性能集成**：与现有 SharedDataPool 无缝集成

### 1.3 技术路线

- **阶段一**：基础 MMS 服务器集成
- **阶段二**：GOOSE/SV 实时通信
- **阶段三**：报告服务与日志
- **阶段四**：客户端能力与网关模式
- **阶段五**：性能优化与高级功能

## 六、API 使用示例

### 6.1 启动 IEC61850 服务器

```cpp
#include "IEC61850Server.h"
#include "IEC61850Mapper.h"

int main() {
    // 创建 Mapper
    IEC61850Mapper::Config mapperConfig;
    mapperConfig.create = true;
    IEC61850Mapper* mapper = IEC61850Mapper::create(mapperConfig);

    // 创建 SharedDataPool
    SharedDataPool* pool = SharedDataPool::create("MyPool", true);

    // 创建 IEC61850 服务器
    IEC61850Server::Config serverConfig;
    serverConfig.modelName = "IPCSharedDataPool";
    serverConfig.configFilename = "model.cfg";
    serverConfig.tcpPort = 102;
    serverConfig.enableLogService = true;

    IEC61850Server* server = IEC61850Server::create(serverConfig);
    server->bindToMapper(mapper);
    server->bindToDataPool(pool);

    // 启动服务器
    server->start();

    // 运行事件循环
    while (running) {
        server->updateFromDataPool();
        sleep(1);
    }

    // 清理
    server->stop();
    server->destroy();
    pool->destroy();
    mapper->destroy();

    return 0;
}
```

### 6.2 使用 GOOSE 发布器

```cpp
// 创建 GOOSE 发布器
GoosePublisherAdapter::Config gooseConfig;
gooseConfig.interfaceName = "eth0";
gooseConfig.vlanPriority = 4;
gooseConfig.vlanId = 100;
gooseConfig.appId = 0x0001;
memcpy(gooseConfig.dstAddress, "\x01\x0C\xCD\x01\x00\x01", 6);
gooseConfig.dataSetRef = "IED1/LD0/LLN0$GOOSE1";

GoosePublisherAdapter* goose = GoosePublisherAdapter::create(gooseConfig);
server->addGoosePublisher(0, goose);

// 更新 GOOSE 值
MmsValue* value = MmsValue_newBoolean(true);
goose->updateValue(0, value);
goose->publish();
MmsValue_delete(value);
```

### 6.3 订阅远程 GOOSE

```cpp
// 订阅回调
auto callback = [](const char* goCBRef, MmsValue* value) {
    printf("GOOSE %s updated: %d\n", goCBRef,
           MmsValue_getBoolean(value));
};

// 订阅 GOOSE
server->subscribeGoose("RemoteIED1/GOOSE1", callback);
```

### 6.4 网关模式

```cpp
// 创建网关
IEC61850Gateway::GatewayConfig gatewayConfig;

// 配置服务器
gatewayConfig.serverConfig.modelName = "GatewayIED";
gatewayConfig.serverConfig.tcpPort = 102;

// 添加客户端连接
IEC61850Client::Config clientConfig1;
clientConfig1.hostname = "ied1.example.com";
clientConfig1.port = 102;
gatewayConfig.clientConfigs.push_back(clientConfig1);

// 添加转发规则
IEC61850Gateway::ForwardRule rule;
rule.sourceRef = "ied1.example.com/IED1/LD0/XCBR1/Pos/stVal";
rule.destRef = "IED1/LD1/XCBR1/Pos/stVal";
rule.direction = ForwardDirection::CLIENT_TO_SERVER;
gatewayConfig.forwardRules.push_back(rule);

// 创建并启动网关
IEC61850Gateway* gateway = IEC61850Gateway::create(gatewayConfig);
gateway->start();
gateway->enableForwarding(true);
```
