### 已知问题和限制

#### 编译问题
由于 libiec61850 API 不兼容，以下模块暂时无法编译：
- IEC61850Server.cpp
- ReportAdapter.cpp
- GooseAdapter.cpp
- SVAdapter.cpp

这些模块在编译时会被跳过（BUILD_IEC61850=OFF）。

#### 需要特殊环境的测试
- GOOSE/SV 适配器测试需要网络接口配置
- 某些测试可能需要 root 权限来操作共享内存
